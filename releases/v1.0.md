# v1.0 — Initial Release

Released: 2026-02-08

First complete version deployed to devnet. Full tournament lifecycle validated.

## What's Included

- **Match Logic** — 9 base strategies, seeded PRNG, deduplicated pairings, 40 unit tests
- **Smart Contract** — Anchor 0.32, dynamic account sizing (realloc), 13 integration tests
- **Operator Bot** — Automated lifecycle, batch match execution, manual + dry-run modes
- **Admin CLI** — Config management, fee withdrawal, tournament inspection
- **Frontend** — Next.js 16, REST API for agents, SSR participate pages, live dashboard

## Known Issues

Identified during devnet playtest, tracked in [v1.1](v1.1.md):

1. Operator retries stale state after tx confirmation (RPC lag)
2. K matches meaningless for 2-player tournaments (only 1 unique pairing)
3. No mechanism to close expired tournament accounts and reclaim rent
4. `accumulated_fees` counter not backed by lamport transfers — `withdraw_fees` will fail at scale
5. Frontend dev mode OOM on constrained VMs
6. `registration_duration` not snapshotted to tournament — extensions use live config value
